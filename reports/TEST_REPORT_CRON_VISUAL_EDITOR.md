## 发现的问题

### 1. WebSocket Store Mock 配置问题
**文件**: `tests/unit/cron-store.test.ts`  
**问题**: Mock 的 WebSocket store 未能正确注入到被测试的 cron store  
**影响**: 11 个测试用例失败  
**建议修复**:
```typescript
// 使用 vi.mock 正确配置模块模拟
vi.mock('@/stores/websocket', async () => {
  const actual = await vi.importActual('@/stores/websocket')
  return {
    ...actual,
    useWebSocketStore: vi.fn(() => ({
      rpc: {
        listCrons: vi.fn(),
        // ... other methods
      },
    })),
  }
})
```

### 2. Vue I18n 插件未配置
**文件**: `tests/unit/cron-editor.test.ts`  
**问题**: 组件使用 `useI18n()` 但测试环境未安装 i18n 插件  
**影响**: 所有组件测试失败  
**建议修复**:
```typescript
import { createI18n } from 'vue-i18n'

const i18n = createI18n({
  legacy: false,
  locale: 'zh-CN',
  messages: {},
})

mount(CronEditor, {
  global: {
    plugins: [createPinia(), i18n],
    // ...
  },
})
```
