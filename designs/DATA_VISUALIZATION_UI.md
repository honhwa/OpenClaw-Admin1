# 数据可视化组件库 UI 设计稿

## 设计概述
为 OpenClaw-Admin 平台设计数据可视化组件库，提供丰富的图表和仪表板组件，帮助用户直观理解数据。

---

## 1. 功能需求

### 1.1 核心组件
- 折线图（趋势分析）
- 柱状图（对比分析）
- 饼图/环形图（占比分析）
- 雷达图（多维度对比）
- 热力图（密度分布）
- 数据卡片（关键指标）
- 进度条/环形进度
- 表格可视化

### 1.2 用户场景
1. 查看任务完成趋势（折线图）
2. 对比不同成员的工作量（柱状图）
3. 查看任务状态分布（饼图）
4. 监控系统性能指标（数据卡片）
5. 分析用户活跃度（热力图）

## 3. 组件规范

### 3.1 颜色规范

| 图表类型 | 主色调 | 辅助色 | 警告色 | 成功色 |
|---------|--------|--------|--------|--------|
| 折线图 | #409EFF | #67C23A | #E6A23C | #F56C6C |
| 柱状图 | #409EFF | #67C23A | #E6A23C | #F56C6C |
| 饼图 | 渐变蓝 | 渐变绿 | 渐变橙 | 渐变红 |
| 数据卡片 | 主色 | - | 橙 | 绿 |

### 3.2 字体规范
- **数值**: 32px, 粗体
- **标题**: 14px, 常规
- **说明**: 12px, 常规，灰色

### 3.3 间距规范
- 卡片内边距：20px
- 卡片间距：16px
- 图表内边距：24px

## 5. 技术实现建议

### 5.1 前端组件结构
```vue
<template>
  <div class="data-visualization">
    <!-- 数据卡片 -->
    <div class="data-cards">
      <data-card
        v-for="card in cards"
        :key="card.id"
        :title="card.title"
        :value="card.value"
        :trend="card.trend"
        :icon="card.icon"
      />
    </div>
    
    <!-- 图表区域 -->
    <div class="charts">
      <line-chart
        :data="trendData"
        :options="lineOptions"
      />
      <pie-chart
        :data="distributionData"
        :options="pieOptions"
      />
    </div>
  </div>
</template>
```

### 5.2 图表库选择
- **推荐**: ECharts (功能丰富，性能好)
- **备选**: Chart.js (轻量级)
- **备选**: AntV G2 (阿里出品)

### 5.3 API 接口设计
```javascript
// 获取仪表板数据
GET /api/dashboard/data
{
  "timeRange": "last_7_days",
  "metrics": ["tasks", "users", "performance"]
}

// 获取趋势数据
GET /api/dashboard/trend
{
  "metric": "tasks_completed",
  "granularity": "daily",
  "startDate": "2026-03-01",
  "endDate": "2026-04-11"
}
```
