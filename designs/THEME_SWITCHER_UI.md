# 多主题切换系统 UI 设计稿

## 设计概述
为 OpenClaw-Admin 平台设计多主题切换系统，支持用户在亮色/暗色/自定义主题之间自由切换，提供个性化的视觉体验。

---

## 1. 功能需求

### 1.1 核心功能
- 亮色主题（默认）
- 暗色主题
- 自动跟随系统
- 自定义主题配置
- 主题预览
- 主题同步（多设备）

### 1.2 用户场景
1. 用户夜间工作切换暗色主题
2. 用户根据偏好自定义主题色
3. 系统自动跟随系统主题
4. 用户快速预览不同主题效果

## 4. 交互设计

### 4.1 主题切换流程
1. 用户点击主题切换按钮
2. 显示主题选择面板
3. 用户选择目标主题
4. 系统应用新主题（带过渡动画）
5. 主题设置保存到本地/云端

### 4.2 自动主题切换
1. 系统检测操作系统主题变化
2. 自动切换到对应主题
3. 显示切换通知（可选）

### 4.3 主题预览
1. 鼠标悬停主题卡片
2. 显示预览效果
3. 点击应用主题

## 5. 技术实现建议

### 5.1 前端实现
```vue
<template>
  <div class="theme-switcher">
    <el-dropdown @command="handleThemeChange">
      <el-button icon="Moon">
        {{ currentThemeLabel }}
      </el-button>
      <el-dropdown-menu>
        <el-dropdown-item command="light">☀️ 亮色主题</el-dropdown-item>
        <el-dropdown-item command="dark">🌙 暗色主题</el-dropdown-item>
        <el-dropdown-item command="auto">💻 自动（跟随系统）</el-dropdown-item>
        <el-dropdown-item divided command="custom">⚙️ 自定义主题</el-dropdown-item>
      </el-dropdown-menu>
    </el-dropdown>
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue';

const currentTheme = ref('light');

// 监听系统主题变化
onMounted(() => {
  const mediaQuery = window.matchMedia('(prefers-color-scheme: dark)');
  mediaQuery.addEventListener('change', (e) => {
    if (currentTheme.value === 'auto') {
      applyTheme(e.matches ? 'dark' : 'light');
    }
  });
});

// 切换主题
const handleThemeChange = (theme) => {
  currentTheme.value = theme;
  applyTheme(theme);
  saveThemePreference(theme);
};

// 应用主题
const applyTheme = (theme) => {
  document.documentElement.setAttribute('data-theme', theme);
};
</script>
```

### 5.2 CSS 变量管理
```css
/* 基础变量 */
:root {
  --transition-duration: 0.3s;
  --transition-timing: ease-in-out;
}

/* 主题过渡 */
* {
  transition: background-color var(--transition-duration) var(--transition-timing),
              color var(--transition-duration) var(--transition-timing),
              border-color var(--transition-duration) var(--transition-timing);
}
```
