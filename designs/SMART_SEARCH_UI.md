# 智能搜索与筛选系统 UI 设计稿

## 设计概述
为 OpenClaw-Admin 平台设计智能搜索与筛选系统的用户界面，提供高效、直观的数据查找体验。

---

## 1. 功能需求

### 1.1 核心功能
- 全局搜索（关键词）
- 高级筛选（多条件组合）
- 日期范围选择
- 保存常用筛选条件
- 搜索历史
- 智能建议/自动补全

### 1.2 用户场景
1. 管理员搜索特定任务名称
2. 筛选出某个时间段内的审计日志
3. 查找特定状态和负责人的任务组合
4. 保存常用的筛选条件供下次使用

## 3. 组件规范

### 3.1 搜索输入框
- **宽度**: 100% 或固定 600px
- **高度**: 40px
- **图标**: 左侧搜索图标，右侧清除按钮
- **占位符**: "搜索任务名称、ID、描述..."
- **交互**: 输入时实时显示建议（防抖 300ms）

### 3.2 高级筛选按钮
- **位置**: 搜索栏右侧
- **样式**: 下拉按钮
- **图标**: 筛选漏斗图标

### 3.3 筛选条件标签
- **样式**: 胶囊形状，带关闭按钮
- **颜色**: 主色调背景，白色文字
- **交互**: 点击关闭按钮移除该条件

### 3.4 日期范围选择器
- **组件**: 双日期选择器
- **格式**: YYYY-MM-DD
- **快捷选项**: 今天、本周、本月、最近 7 天、最近 30 天

## 4. 交互流程

### 4.1 搜索流程
1. 用户在搜索框输入关键词
2. 输入停止 300ms 后显示建议
3. 用户选择建议或直接回车
4. 显示搜索结果，高亮关键词
5. 支持分页浏览

### 4.2 高级筛选流程
1. 点击"高级筛选"按钮
2. 展开筛选面板
3. 选择/输入筛选条件
4. 点击"应用筛选"或"重置"
5. 列表自动更新显示筛选结果

### 4.3 保存筛选流程
1. 设置好筛选条件后点击"保存筛选"
2. 弹出命名对话框
3. 输入筛选名称
4. 保存到常用筛选列表
5. 下次可直接从下拉选择

## 5. 技术实现建议

### 5.1 前端组件结构
```vue
<template>
  <div class="smart-search">
    <div class="search-bar">
      <el-input
        v-model="keyword"
        placeholder="搜索任务名称、ID、描述..."
        prefix-icon="Search"
        clearable
        @input="handleSearch"
      />
      <el-button @click="showAdvancedFilter = !showAdvancedFilter">
        高级筛选
      </el-button>
    </div>
    
    <div v-if="showAdvancedFilter" class="advanced-filter">
      <!-- 筛选条件表单 -->
    </div>
    
    <div class="filter-tags">
      <el-tag
        v-for="filter in activeFilters"
        :key="filter.key"
        closable
        @close="removeFilter(filter)"
      >

      </el-tag>
    </div>
  </div>
</template>
```

### 5.3 智能建议实现
```javascript
// 防抖搜索
const debounce = (fn, delay) => {
  let timer = null;
  return (...args) => {
    clearTimeout(timer);
    timer = setTimeout(() => fn(...args), delay);
  };
};

// 搜索建议
const searchSuggestions = debounce(async (keyword) => {
  if (keyword.length < 2) return;
  const suggestions = await api.getSearchSuggestions(keyword);
  showSuggestions(suggestions);
}, 300);
```
