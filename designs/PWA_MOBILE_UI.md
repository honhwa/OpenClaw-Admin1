# 移动端 PWA 界面设计稿

## 设计概述
为 OpenClaw-Admin 平台设计移动端 PWA（Progressive Web App）界面，提供原生应用般的移动体验，支持离线使用和推送通知。

---

## 1. 功能需求

### 1.1 核心功能
- 响应式布局
- 离线缓存
- 推送通知
- 添加到主屏幕
- 手势导航
- 移动端优化组件

### 1.2 用户场景
1. 用户在手机上查看任务列表
2. 离线时查看缓存数据
3. 接收任务分配推送通知
4. 将 PWA 添加到手机主屏幕
5. 使用手势快速导航

---

## 3. 手势导航

### 3.1 支持的手势

| 手势 | 操作 | 场景 |
|------|------|------|
| 左滑 | 返回上一页 | 所有页面 |
| 右滑 | 返回上一页 | 所有页面 |
| 下拉 | 刷新数据 | 列表页面 |
| 上滑 | 加载更多 | 列表页面 |
| 长按 | 进入多选模式 | 列表项 |
| 双指捏合 | 缩放图表 | 图表页面 |

### 3.2 手势反馈
- 左滑/右滑：显示页面预览
- 下拉刷新：显示加载动画
- 长按：震动反馈

## 6. 技术实现建议

### 6.1 PWA 配置
```javascript
// manifest.json
{
  "name": "OpenClaw Admin",
  "short_name": "OpenClaw",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#1A1A2E",
  "theme_color": "#409EFF",
  "icons": [
    {
      "src": "/icon-192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "/icon-512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ]
}
```

### 6.2 Service Worker
```javascript
// sw.js
self.addEventListener('install', (event) => {
  event.waitUntil(
    caches.open('openclaw-v1').then((cache) => {
      return cache.addAll([
        '/',
        '/index.html',
        '/manifest.json',
        '/icon-192.png',
        '/icon-512.png'
      ]);
    })
  );
});

self.addEventListener('fetch', (event) => {
  event.respondWith(
    caches.match(event.request).then((response) => {
      return response || fetch(event.request);
    })
  );
});
```

### 6.3 响应式布局
```css
/* 移动端优先 */
.container {
  padding: 16px;
}

/* 平板适配 */
@media (min-width: 768px) {
  .container {
    padding: 24px;
    max-width: 720px;
    margin: 0 auto;
  }
}

/* 桌面端适配 */
@media (min-width: 1024px) {
  .container {
    max-width: 1200px;
  }
}
```
