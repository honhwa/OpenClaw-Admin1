## 第二部分：交互说明

### 2.1 页面交互

#### 页面加载动画

```
加载状态:
- 显示骨架屏 (Skeleton)
- 骨架屏颜色：#F5F7FA
- 闪烁动画：2s 循环
- 加载完成：淡入效果 (300ms)
```

#### 页面切换动画

```
进入动画:
- 方向：从右向左
- 时长：300ms
- 缓动：cubic-bezier(0.4, 0, 0.2, 1)

退出动画:
- 方向：向左淡出
- 时长：200ms
```

### 2.2 组件交互

#### 批量操作交互

**选择记录**:
1. 点击复选框 → 选中单条记录
2. 点击全选框 → 选中当前页所有记录
3. 选中记录数 > 0 → 显示批量操作栏
4. 批量操作栏：从顶部滑入 (300ms)

**批量删除**:
1. 点击"批量删除"按钮
2. 弹出确认对话框 (居中淡入)
3. 显示选中记录列表 (最多 5 条，超出显示"等 N 条")
4. 点击"确认删除" → 执行删除 → 显示成功提示 → 刷新列表
5. 点击"取消" → 关闭对话框

**批量状态变更**:
1. 点击"批量状态变更"下拉按钮
2. 展开状态选项菜单
3. 选择目标状态
4. 弹出确认对话框
5. 确认后执行变更 → 显示成功/失败统计

#### 搜索交互

**搜索输入**:
1. 点击搜索框 → 聚焦，显示边框高亮
2. 输入关键词 → 防抖 300ms 后显示建议
3. 搜索建议列表：最多显示 5 条
4. 选择建议或回车 → 执行搜索
5. 显示搜索结果，高亮关键词

**高级筛选**:
1. 点击"高级筛选"按钮
2. 筛选面板从上方滑下 (300ms)
3. 选择/输入筛选条件
4. 点击"应用筛选" → 列表更新
5. 点击"重置" → 清空所有条件
6. 点击外部区域 → 面板收起

**筛选标签**:
1. 应用筛选后，在搜索框下方显示筛选标签
2. 点击标签关闭按钮 → 移除该条件
3. 点击"清除全部" → 移除所有筛选
4. 无筛选条件时，标签区域隐藏

#### 图表交互

**悬停提示**:
1. 鼠标悬停数据点
2. 显示 Tooltip 提示框
3. 显示详细数据信息
4. 鼠标移开 → Tooltip 淡出

**图例点击**:
1. 点击图例项
2. 对应数据系列高亮/隐藏
3. 图表重新渲染
4. 再次点击恢复

**时间范围切换**:
1. 点击时间范围按钮 (日/周/月/年)
2. 按钮高亮显示当前选择
3. 图表数据平滑过渡更新
4. 动画时长：500ms

#### 弹窗交互

**弹窗打开**:
1. 触发弹窗事件
2. 背景遮罩淡入 (300ms)
3. 弹窗从中心缩放出现 (scale 0.95 → 1, 300ms)
4. 焦点 trap 到弹窗内

**弹窗关闭**:
1. 点击关闭按钮/取消按钮/背景遮罩
2. 弹窗缩放淡出 (300ms)
3. 背景遮罩淡出 (300ms)
4. 焦点返回触发元素

**拖拽弹窗** (可选):
1. 拖拽弹窗标题栏
2. 弹窗跟随鼠标移动
3. 边界限制：不超出可视区域

### 2.3 反馈交互

#### 加载状态

```
按钮加载:
- 显示 loading 图标
- 图标旋转动画 (1s 循环)
- 按钮禁用，防止重复点击

页面加载:
- 显示骨架屏
- 骨架条闪烁动画

数据加载:
- 列表项逐条淡入
- 每条间隔 50ms
```

#### 成功提示

```
位置：右上角固定
样式：绿色背景 + 对勾图标
时长：3s 后自动消失
动画：从右侧滑入
```

#### 错误提示

```
位置：右上角固定
样式：红色背景 + 感叹号图标
时长：5s 后自动消失
动画：从右侧滑入
```

#### 确认提示

```
位置：居中弹窗
样式：白色背景 + 图标
按钮：取消 (次要) + 确认 (主要/危险)
动画：淡入 + 缩放
```

### 2.4 移动端手势交互

#### 基础手势

**左滑返回**:
- 触发区域：屏幕左侧 20px
- 手势：从左向右滑动
- 反馈：当前页面右移，返回页面无缝衔接

**下拉刷新**:
- 触发区域：列表顶部
- 手势：向下拉动超过 80px
- 反馈：显示刷新图标 + "松开刷新"提示
- 刷新中：显示 loading 动画
- 完成：显示成功提示，列表滚动到顶部

**上拉加载**:
- 触发区域：列表底部
- 手势：滚动到底部自动触发
- 反馈：显示"加载中..."提示
- 无更多数据：显示"没有更多了"

**长按操作**:
- 触发：长按列表项 500ms
- 反馈：震动反馈 + 弹出操作菜单
- 菜单项：编辑、删除、分享

## 第四部分：开发指南

### 4.1 技术栈建议

```
框架：Vue 3 + TypeScript
UI 组件库：Element Plus / Ant Design Vue
图表库：ECharts
状态管理：Pinia
路由：Vue Router
HTTP 客户端：Axios
样式方案：SCSS + CSS Variables
```

### 4.2 项目结构

```
src/
├── assets/          # 静态资源
│   ├── icons/       # 图标
│   ├── images/      # 图片
│   └── styles/      # 样式文件
│       ├── variables.scss    # 变量
│       ├── mixins.scss       # 混入
│       └── global.scss       # 全局样式
├── components/      # 通用组件
│   ├── Button/
│   ├── Input/
│   ├── Table/
│   ├── Modal/
│   └── ...
├── views/           # 页面组件
│   ├── BatchOperation/
│   ├── SmartSearch/
│   ├── Dashboard/
│   ├── Permission/
│   ├── Theme/
│   └── ...
├── composables/     # 组合式函数
├── stores/          # 状态管理
├── router/          # 路由配置
└── api/             # API 接口
```
