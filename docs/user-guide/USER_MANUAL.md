## 快速入门

### 系统要求

- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **网络**: 需要访问 OpenClaw Gateway 服务
- **账户**: 需要管理员创建的账户

### 首次使用

1. **访问系统**
   - 打开浏览器，访问 `http://your-server:3001`
   - 或使用 HTTPS: `https://your-domain`

2. **登录**
   - 输入管理员提供的用户名和密码
   - 点击"登录"按钮

3. **查看仪表盘**
   - 登录后自动进入仪表盘
   - 查看系统运行状态和关键指标

## 登录与认证

### 登录流程

1. 访问登录页面
2. 输入用户名和密码
3. 点击"登录"
4. 系统验证凭据并创建会话
5. 自动跳转到仪表盘

### 会话管理

- **会话有效期**: 默认 24 小时
- **自动续期**: 活跃会话自动续期
- **多设备登录**: 支持多设备同时登录
- **安全退出**: 点击"退出登录"清除会话

### 密码安全

- **密码要求**: 至少 8 位，包含大小写字母和数字
- **密码加密**: 使用 PBKDF2 加密存储
- **传输安全**: 所有通信使用 HTTPS

## 仪表盘

### 运行总览

仪表盘显示系统关键指标:

- **活跃会话数**: 当前在线的 AI 会话数量
- **Token 使用量**: 今日/本周/本月 Token 消耗
- **智能体数量**: 已配置的 AI 智能体总数
- **消息吞吐量**: 实时消息处理速率

### Token 使用趋势

- **图表展示**: 折线图显示 Token 使用趋势
- **时间范围**: 可选择今日/本周/本月
- **智能体对比**: 对比不同智能体的 Token 使用

### 会话活跃度

- **活跃会话**: 最近 1 小时有活动的会话
- **会话时长**: 平均会话持续时间
- **消息数量**: 会话中交换的消息总数

### 实时事件流

- **事件类型**: 会话创建、消息发送、工具调用等
- **实时更新**: WebSocket 推送实时事件
- **事件筛选**: 按类型/智能体/时间筛选

### Top 统计

- **Top 模型**: 使用量最高的模型排名
- **Top 渠道**: 消息来源渠道排名
- **Top 工具**: 调用次数最多的工具排名

## 会话管理

### 创建会话

1. 点击"新建会话"按钮
2. 选择 AI 智能体
3. 选择模型 (可选)
4. 输入初始消息 (可选)
5. 点击"创建"

### 会话列表

**查看会话**:
- 所有活跃会话列表
- 按创建时间/最后活动排序
- 支持搜索和筛选

**会话筛选**:
- 按智能体筛选
- 按渠道筛选 (QQ/飞书/钉钉等)
- 按状态筛选 (活跃/空闲/已结束)
- 按时间范围筛选

### 会话详情

**会话信息**:
- 会话 ID
- 关联智能体
- 创建时间
- 最后活动时间
- Token 使用量
- 消息数量

**消息记录**:
- 查看所有消息历史
- 消息时间戳
- 发送者/接收者
- Token 消耗详情

### 会话操作

**重置会话**:
- 清除会话上下文
- 保留历史记录
- 重新开始对话

**删除会话**:
- 永久删除会话
- 删除所有消息记录
- 需要确认操作

**导出会话**:
- 导出为 JSON 格式
- 包含所有消息和元数据
- 用于备份或分析

## Cron 任务管理

### 任务列表

查看所有定时任务:
- 任务名称
- Cron 表达式
- 执行命令
- 当前状态
- 最后执行时间
- 下次执行时间

### 创建任务

**方式一: 手动输入**
1. 点击"新建任务"
2. 输入任务名称
3. 输入 Cron 表达式
4. 输入执行命令
5. 添加描述 (可选)
6. 点击"创建"

**方式二: 使用模板**
1. 点击"新建任务"
2. 选择预设模板
3. 修改命令和参数
4. 点击"创建"

**方式三: 可视化编辑器**
1. 点击"可视化编辑"
2. 选择时间频率 (分钟/小时/天/周/月)
3. 设置具体时间
4. 系统自动生成 Cron 表达式
5. 输入命令和描述
6. 点击"创建"

### Cron 表达式参考

| 字段 | 允许值 | 特殊字符 |
|------|--------|----------|
| 分钟 | 0-59 | , * - / |
| 小时 | 0-23 | , * - / |
| 日期 | 1-31 | , * - / ? |
| 月份 | 1-12 | , * - / |
| 星期 | 0-6 | , * - / ? |

**常用表达式**:
- `* * * * *` - 每分钟
- `0 * * * *` - 每小时
- `0 0 * * *` - 每天凌晨
- `0 8 * * *` - 每天早上 8 点
- `0 3 * * 0` - 每周日凌晨 3 点
- `0 9 1 * *` - 每月 1 号上午 9 点

### 任务操作

**启用/禁用**:
- 切换任务开关
- 禁用任务不会执行
- 保留任务配置

**立即执行**:
- 不等待 cron 时间
- 立即执行任务
- 查看执行结果

**编辑任务**:
- 修改任务配置
- 更新 Cron 表达式
- 保存更改

**删除任务**:
- 永久删除任务
- 删除执行历史
- 需要确认操作

### 执行历史

**查看历史**:
- 所有执行记录
- 执行时间
- 执行状态 (成功/失败)
- 执行输出
- 错误信息
- 执行时长

**筛选历史**:
- 按任务筛选
- 按状态筛选
- 按时间范围筛选

**清理历史**:
- 删除旧记录
- 保留最近 N 条
- 按时间清理

### 模板管理

**内置模板**:
- 每分钟
- 每小时
- 每天凌晨
- 每天 8 点
- 每周日 3 点
- 每月 1 号 9 点

**自定义模板**:
- 创建常用任务模板

## 常见问题

### 登录问题

**Q: 无法登录怎么办？**
- 检查用户名和密码是否正确
- 确认网络连接正常
- 清除浏览器缓存
- 联系管理员重置密码

**Q: 会话过期怎么办？**
- 重新登录即可
- 系统会自动续期活跃会话
- 检查浏览器 Cookie 设置

### 任务管理问题

**Q: 任务没有按时执行？**
- 检查 Cron 表达式是否正确
- 确认任务状态为"启用"
- 查看执行历史是否有错误
- 检查服务器时间是否正确

**Q: 如何调试任务？**
- 使用"立即执行"功能手动触发
- 查看执行输出的日志
- 检查命令是否可执行
- 验证环境变量设置

### 模型问题

**Q: 模型连接失败？**
- 检查 API Key 是否正确
- 验证端点 URL 是否可访问
- 检查网络防火墙设置
- 查看模型提供商状态

**Q: Token 使用异常？**
- 检查是否有异常会话
- 查看 Token 使用趋势
- 限制单条消息 Token 数
- 设置 Token 使用上限

### 性能问题

**Q: 页面加载慢？**
- 检查网络连接
- 清除浏览器缓存
- 减少仪表盘数据量
- 联系管理员优化服务器

**Q: WebSocket 连接不稳定？**
- 检查网络稳定性
- 刷新页面重新连接
- 检查防火墙设置
- 联系管理员检查服务器
